## 사용 시 주의

1. 회사마다 같은 한국어에 다른 영어를 쓸 수 있음 → glossary가 IFRS보다 우선
2. 복합 용어는 부분 매칭으로: "당기손익-공정가치측정금융자산" = 여러 단어 조합
3. 축약형 주의: FVTPL, FVOCI, OCI 등은 그대로 사용 가능
4. 단수/복수 구분: "Receivable" vs "Receivables" — DOCX 기존 표현 따름
